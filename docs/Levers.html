Levers

Levers

new Levers(fileNameopt, optsopt)

Source:
  • index.js, line 50

Create a new Levers instance and access or create the data file. By default the file is stored in the OS application data directory. For example, on Windows this is C:\Users\YourName\AppData\YourApp. Levers creates a subdirectory data that will contain the file.

To override this default directory, pass the dir property in the options object. In both this and the above case, the directory and file will be created if they do not exist already.

If the file does exist already, it will be read into memory.

Example
import Levers from 'levers'

 // creates a file named `.settings.json` in the default directory
 const settings = new Levers()

 // creates `app.json` in the default directory
 const appSettings = new Levers('app')

 // creates `config.json` in custom directory
 // uses opts.defaults to set initial values
 const path = require('path')
 const windowSettings = new Levers('config', {
   dir: path.resolve(__dirname, 'settings')
   defaults: { dev: true, theme: 'light' }
 })

del(key)

Remove a specified key from the store

Example
levers.set('dev', false)
levers.set({ ui: { theme: 'blue' } })

levers.has('dev')
// -> true
levers.has('ui.theme')
// -> true

levers.del('dev')
levers.del('ui.theme')

levers.has('dev')
// -> false
levers.has('ui.theme')
// -> false
